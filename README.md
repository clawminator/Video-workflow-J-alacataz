# Video-workflow-J-alacataz

## Want to try this ComfyUI Wan 2.2 SVI Pro 2 workflow?

Anybody want to try out my **ComfyUI Wan 2.2 SVI Pro 2** workflow?

- Generation time: about **8 minutes** for a **26-second** video at **768 x 512** on an **RTX 5090**.
- It may also work on a **4090** (YMMV).

### Files in this repo

- Workflow image: `Video_00020.png`
- Reference image: `ComfyUI_00063.png`

### Tips for using your own reference image

When trying your own reference image, be sure to clearly show the faces and/or bodies of your actors.

> **I'm sure someone out there is more creative than me. Enjoy!**

### Optional speed-up patch (RIFE VFI)

To speed up RIFE VFI, apply this patch:

https://github.com/Fannovel16/ComfyUI-Frame-Interpolation/pull/102

### Quick start (new users)

1. Clone or download this repository.
2. Open ComfyUI and load the workflow shown in `Video_00020.png`.
3. Start with the provided reference setup from `ComfyUI_00063.png`.
4. If you switch to your own image, make sure faces/bodies are clearly visible.
5. (Optional) Apply the RIFE VFI patch above for faster frame interpolation.
6. Run the workflow and adjust prompts/settings to your taste.
